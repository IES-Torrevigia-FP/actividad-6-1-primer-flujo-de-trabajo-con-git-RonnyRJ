# Notas de Git – Actividad 6.1

- Concepto 1: Directorio de trabajo: Carpeta donde estamos editando los archivos. Git observa los cambios que hacemos aqui.
- Concepto 2: Zona de staging: Area donde ponemos los archivos que se quiere incluir en el proximo commit
- Concepto 3: Historial de commits: Registro de todos los commits hechos en el repositorio.
